Documentación Técnica: SharkCorp Portfolio

`Versión: 1.1.0`

`Focus: Showcase de Ingeniería & Talento Humano`

## 1. Concepto Actualizado

**SharkCorp** se presenta como un estudio boutique de ingeniería web. La web debe funcionar como una vitrina de alta gama donde los proyectos son los protagonistas y el equipo de 3 programadores representa el núcleo de experiencia de la empresa.

## 2. Secciones Principales

1. **Home / Hero:** Introducción breve y directa.
2. **Catálogo de Proyectos:** Galería visual con filtros por tecnología.
3. **The Squad (Stack Humano):** Perfiles detallados de los 3 desarrolladores.
4. **Stack Tecnológico:** Las herramientas que domina el equipo.
5. **Contacto:** Enlace directo para nuevos proyectos.

## 3. Especificaciones del Equipo (Los 3 de SharkCorp)

Para que el diseño sea coherente, definimos estos roles:

* **Dev 1 (Fullstack Lead):** Especialista en Arquitectura y Astro.
* **Dev 2 (Frontend Magician):** Especialista en React y Animaciones.
* **Dev 3 (Backend & Cloud):** Especialista en Node.js, Go y AWS.

## 4. Stack Tecnológico de la Web

* **Framework:** Astro 4.x (SSG).
* **Gestión de Proyectos:** Content Collections (archivos `.md` para cada proyecto).
* **Estilos:** Tailwind CSS (Dark Theme).
* **Componentes:** React para el filtrado del catálogo y efectos de hover.
